# Nagada Dart Reference Client

This folder contains a minimal reference Dart client for Nagada and a small example.

Run the example:

```bash
cd reference/client/dart
dart pub get
dart run example/simple_console_app.dart
```

Notes:
- The package name in `pubspec.yaml` is `nagada_client` for local reference.
- The example imports the library as `package:nagada_client/nagada_client.dart`.
