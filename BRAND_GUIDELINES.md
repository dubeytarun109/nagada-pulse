## Brand Identity

### The Name

**Nagada** = The Synchronization Protocol Standard (specification)  
**Nagada-Pulse** = The Heartbeat-Based Implementation Model and Reference Project

Use "Nagada" when referring to the protocol specification itself. Use "Nagada-Pulse" when discussing the heartbeat model, implementation, or the project organization.
